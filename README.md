
# any-crosshair

使用任意PNG作为屏幕准心

## 使用方法

1. 下载exe 到任意目录
2. 添加复制要使用的PNG图片到exe 同目录, 并修改为default.png
3. 双击启动

### 快捷键使用方法

默认注册F1-F12 快捷键，目前不支持修改，注册不上的不会使用
如按下快捷键F1时，自动读取同目录F1.png
```text
.
 \_ any-crosshair.exe
 \_ default.png
 \_ F1.png
 \_ F12.png
```

# 准心制作

普通准心制作可使用[https://crosshair.themeta.gg/](https://crosshair.themeta.gg/)


其他自定义准心可使用[http://ps.bimant.com/](http://ps.bimant.com/) ，
只要能制作透明PNG 均可
